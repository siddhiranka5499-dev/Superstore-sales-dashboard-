
📊 SuperStore Sales Dashboard

📌 Project Overview

The SuperStore Sales Dashboard is an interactive Power BI project designed to analyze sales performance, customer purchasing behavior, and business trends. The dashboard transforms raw sales data into meaningful visual insights using interactive charts, KPIs, and slicers, enabling users to monitor business performance and support data-driven decision-making. The data was cleaned and prepared in Excel before being imported into Power BI. :contentReference[oaicite:0]{index=0}

🎯 Project Objectives

- Analyze overall sales performance.
- Monitor customer purchasing trends.
- Identify top-performing states.
- Compare sales across different age groups and genders.
- Track monthly sales and order trends.
- Analyze order status across different sales channels.
- Provide interactive filtering for better business analysis.
